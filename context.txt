## Việc đang làm

Build data layer cho đồ án Multi-Agent chatbot tư vấn nghề nghiệp Hà Nội.
Đọc file data/job_details.json (14,833 jobs HN từ topcv) → đẩy vào:
  - SQLite (jobs + companies + phuongs + loai_jobs + M:N + FTS5 title)

Phạm vi chốt: chỉ Pipeline 0 + 3 + 4. Bỏ:
  - Pipeline 1 (skill extract) — block do canonical_skills.json chưa có
  - Pipeline 2 (work_mode extract) — làm task sau

Refs đã đọc: docs/design_data.md (schema spec), docs/09-job-search.md, docs/07-skill-gap.md.


## Stack đã chốt

- SQLite (built-in FTS5)
- Qdrant Cloud — env vars: QDRANT_URL, QDRANT_API_KEY
- Embedding: BAAI/bge-m3 chạy LOCAL qua transformers (KHÔNG dùng sentence-transformers, KHÔNG dùng LangChain wrapper)
    from transformers import AutoTokenizer, AutoModel
    tokenizer = AutoTokenizer.from_pretrained("BAAI/bge-m3")
    model = AutoModel.from_pretrained("BAAI/bge-m3")
  Lấy vector dense = CLS token (output[0][:, 0]) + L2 normalize
  Vector size: 1024 (KHÔNG phải 1536 như spec gốc — đổi do dùng bge-m3)
- Cross-platform: pathlib, không hardcode path, auto-detect device (cuda > mps > cpu)
- Conda env: tên thay đổi theo máy
    * Mac: "thesis" (CLAUDE.md ghi)
    * Windows hiện tại: "code"

## Decisions chốt khác

- Cột để NULL ở P0 (Pipeline 1/2 fill sau):
    jobs.work_mode_extracted, jobs.work_mode_extracted_at,
    jobs.skills_extracted_at, bảng job_skills (empty)
- User-side tables (users/conversations/messages/user_profile/memory_facts):
  tạo schema rỗng (app sau dùng được liền, không cần migration)
- Idempotency P0: INSERT OR IGNORE strict, rowcount==0 thì skip junction
- Idempotency P3/P4: query Qdrant scroll lấy existing point IDs, skip jobs đã embed
- Qdrant collection: check-then-create (KHÔNG recreate, tránh xoá data cũ)
- Point ID Qdrant = jobs.id (integer)
- Payload Qdrant = {"job_id": int} only (per spec — KHÔNG dùng langchain-qdrant vì nó bọc thành {page_content, metadata})
- FTS5 sync qua trigger + final rebuild('jobs_fts')
- Dedup: theo link_job, union xa_phuong + loai_job, giữ scalar non-null đầu tiên
- Reject record thiếu 1 trong: link_job, title, company_name, job_description, requirements
